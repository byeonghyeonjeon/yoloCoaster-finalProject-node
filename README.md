# yoloCoaster - 여행 커뮤니티 사이트 - node
### 개요
- 대덕인재개발원에서 진행한 최종 프로젝트 (2018.06.15 ~ 2018.07.16)
- 채팅 및 일정공유 기능

### 기능 소개
- 실시간 기능을 제공하기 위하여 Node.js 서버를 추가적으로 이용 (기본 톰캣)
- socket.io를 이용하여 실시간 채팅 기능 제공
- fullCalendar와 socket.io를 이용하여 실시간 일정 공유 기능 제공
